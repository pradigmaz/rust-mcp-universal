# Closed Plans

Здесь хранятся закрытые планы, чеклисты и roadmap-документы, которые больше не считаются активными рабочими файлами.

Правила:

- новый файл попадает сюда только после явного закрытия всех checklist/blocker-пунктов;
- имя должно содержать дату закрытия и понятный slug;
- активная рабочая версия может оставаться в исходном месте как указатель, но canonical closed copy хранится здесь.
